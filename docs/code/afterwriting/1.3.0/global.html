BaseSectionPresenter

Source:
  • theme/aw-bubble/presenter/base-section-view-presenter.js, line 13
Base presenter for a main section of a plugin. It provides 3 hook methods: * activate() called when section is shown; it will automatically show the view for the presenter * deactivate() called when section is hidden; it will automatically hide the view for the presenter * _scriptChanged invoked when script changes and section is active; script is parsed before calling the hook
